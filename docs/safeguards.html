Data Safeguards

This project is dummy-data-only. No private, protected, or production information belongs in this repository.

NO

Never Commit Sensitive Data

No real user data, no production exports, no credentials, and no internal HC infrastructure details.

ONLY

Use Approved Inputs

Anonymized screenshots, schema metadata, and clearly fake sample data are allowed for build and testing.

CHECK

Review Before Every Commit

Redact screenshots, remove endpoints, and verify no secrets or PII are present in staged files.

Allowed Content

  • Anonymized screenshots
  • Field names, types, relationships
  • Dummy or randomized datasets
  • Application code, tests, and architecture docs

Prohibited Content

  • Production database exports
  • Real names, emails, identifiers
  • Internal HC URLs or IPs
  • API keys, passwords, access tokens
