## What Goes Here

This folder contains the **modern replacement** application built during the hackathon.

### `src/`
Application source code — the modernized version of the Lotus Notes app.

### `dummy-data/`
Randomized/fake test data for development:
- Generated datasets that match the schema structure
- Obviously fake values (e.g., "Jane Doe", "test@example.com", "123 Fake Street")
- Seed scripts or JSON/CSV files

### `docs/`
Planning and architecture documentation:
- **requirements.md** — AI-generated requirements from the LotusApp documentation
- **architecture.md** — Technical architecture decisions
- **conversion-plan.md** — Step-by-step migration plan
- **decisions.md** — Key decisions and trade-offs made during the hackathon
