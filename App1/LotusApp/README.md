All screenshots must be anonymized. No production data. No PII. See [DATA-SAFEGUARDS.md](../../DATA-SAFEGUARDS.md).

---

## What Goes Here

This folder documents the **existing** Lotus Notes application to enable AI-assisted modernization.

### `screenshots/`
Anonymized screenshots of the application UI — forms, views, navigation, dialogs.
- Redact any real names, emails, case numbers, or sensitive data before adding
- Name files descriptively: `main-form.png`, `list-view.png`, `workflow-dialog.png`

### `data-schema/`
Database schema documentation — fields, types, relationships.
- Export or manually document the Notes database schema
- Include field names, data types, validation rules
- **NO real data values** — only structure

### `documentation/`
AI-generated and human-written documentation of the app:
- Business rules and workflows
- User roles and permissions
- Integration points
- Known issues or limitations
